Medical MathPad 3.0 readme1st time MathPad users:     If you are a first time MathPad user that you may just install the file MedMathPad.pdb.  If you already have records installed in MathPad, then you should not install this file, as it will overwrite any records previously installed.MathPad users with records already installed:     If you have been using MathPad and have records installed already, then you should use one of the MathPad install applications to install the MedMathPad.txt file.  This will add the Medical MathPad equations to your previously installed MathPad records.
